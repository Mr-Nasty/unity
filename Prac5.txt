1. SURFACE EFFECTOR
- GameObject → 2D Object → Sprite → Square
- GameObject → 2D Object → Sprite → Circle
- Select Square
- Add Component → Box Collider 2D
    → Tick: Used By Effector
- Add Component → Surface Effector 2D
    → Speed = 5
- Select Circle
- Add Component → Circle Collider 2D
- Add Component → Rigidbody 2D
 
 
2. PLATFORM EFFECTOR
- GameObject → 2D Object → Sprite → Square
- Select Square
- Add Component → Box Collider 2D
    → Tick: Used By Effector   ← must enable this or it won't work
- Add Component → Platform Effector 2D
    → Surface Arc = 180
 
 
3. AREA EFFECTOR
Area Effector pushes objects in a direction.
Force Angle: 0 = right | 90 = up | 180 = left | 270 = down
 
- GameObject → 2D Object → Sprite → Square
- Select Square
- Add Component → Box Collider 2D
    → Tick: Is Trigger
    → Tick: Used By Effector   ← must enable this or it won't work
- Add Component → Area Effector 2D
    → Force Angle = 90         (90 = pushes upward)
    → Force Magnitude = 10
- Select Circle
- Make sure it has: Rigidbody 2D + Circle Collider 2D
 
 
4. PLAYER MOVEMENT SCRIPT (Jump & Move)
- Create script: JumpnMove
- Attach to Circle object
 
Unity 2021 and older:
 
using UnityEngine;
public class JumpnMove : MonoBehaviour
{
    public float jumpForce = 5f;
    public float speed = 5f;
    Rigidbody2D rb;
    void Start()
    {
        rb = GetComponent<Rigidbody2D>();
    }
    void Update()
    {
        float moveX = Input.GetAxis("Horizontal");
        rb.velocity = new Vector2(moveX * speed, rb.velocity.y);
        if (Input.GetKeyDown(KeyCode.Space))
        {
            rb.velocity = new Vector2(rb.velocity.x, jumpForce);
        }
    }
}
 
Unity 2022 and newer (use linearVelocity instead):
 
using UnityEngine;
public class JumpnMove : MonoBehaviour
{
    public float jumpForce = 5f;
    public float speed = 5f;
    Rigidbody2D rb;
    void Start()
    {
        rb = GetComponent<Rigidbody2D>();
    }
    void Update()
    {
        float moveX = Input.GetAxis("Horizontal");
        rb.linearVelocity = new Vector2(moveX * speed, rb.linearVelocity.y);
        if (Input.GetKeyDown(KeyCode.Space))
        {
            rb.linearVelocity = new Vector2(rb.linearVelocity.x, jumpForce);
        }
    }
}
 
- Select Circle
- Make sure it has: Rigidbody 2D + Circle Collider 2D
Controls: A/D or Arrow Keys = move | Space = jump
 
 
5. BUOYANCY EFFECTOR
- GameObject → 2D Object → Sprite → Square
- Select Square
- Add Component → Box Collider 2D
    → Tick: Is Trigger
    → Tick: Used By Effector   ← must enable this or it won't work
- Add Component → Buoyancy Effector 2D
    → Density = 1
    → Linear Drag = 3
    → Angular Drag = 1
- Select Circle
- Make sure it has: Rigidbody 2D + Circle Collider 2D
 
 
6. SOUND TRIGGER EFFECTOR
- GameObject → 2D Object → Sprite → Square
- Rename: SoundZone
- Select SoundZone
- Add Component → Box Collider 2D
    → Tick: Is Trigger
- Add Component → Audio Source
    → Untick: Play On Awake
    → Drag your audio clip into: AudioClip field
- Create script: AudioEffector2D
- Attach to SoundZone
 
using UnityEngine;
public class AudioEffector2D : MonoBehaviour
{
    public AudioSource audioSource;
    private int playerCount = 0;
    void Awake()
    {
        if (audioSource == null)
            audioSource = GetComponent<AudioSource>();
    }
    void OnTriggerEnter2D(Collider2D other)
    {
        if (!other.CompareTag("Player")) return;
        playerCount++;
        if (playerCount == 1 && !audioSource.isPlaying)
            audioSource.Play();
    }
    void OnTriggerExit2D(Collider2D other)
    {
        if (!other.CompareTag("Player")) return;
        playerCount--;
        if (playerCount <= 0)
        {
            playerCount = 0;
            audioSource.Stop();
        }
    }
}
 
- Select Circle (player)
- In Inspector: Tag → Player
  (If Player tag missing: Add Tag → type Player → save → set it)
When player enters zone → sound plays
When player leaves zone → sound stops
