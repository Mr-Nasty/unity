FINDING DISTANCE TO TARGET
===========================
 
Step 1 — Create Project
- Open Unity Hub
- Create project: Universal 3D
 
Step 2 — Create Objects
- GameObject → 3D Object → Cube
- GameObject → 3D Object → Sphere
- Rename:
    Cube   → Player
    Sphere → Target
 
Step 3 — Create Script
- In Project panel: Right Click → Create → C# Script
- Name it: RangeChecker
 
Step 4 — Script Code
 
using UnityEngine;
public class RangeChecker : MonoBehaviour
{
    [SerializeField] Transform target;
    [SerializeField] float range = 5f;
    [SerializeField] float moveSpeed = 5f;
    private bool targetWasInRange = false;
 
    void Update()
    {
        // Player movement with arrow keys
        float moveX = Input.GetAxis("Horizontal");
        float moveZ = Input.GetAxis("Vertical");
        Vector3 movement = new Vector3(moveX, 0f, moveZ);
        transform.Translate(movement * moveSpeed * Time.deltaTime, Space.World);
 
        // Distance calculation
        float distance = Vector3.Distance(target.position, transform.position);
 
        if (distance <= range && !targetWasInRange)
        {
            Debug.LogFormat("Target {0} entered range!", target.name);
            targetWasInRange = true;
        }
        else if (distance > range && targetWasInRange)
        {
            Debug.LogFormat("Target {0} exited range!", target.name);
            targetWasInRange = false;
        }
    }
}
 
Step 5 — Attach Script
- Select Player (Cube)
- Add Component → RangeChecker
 
Step 6 — Assign Target
- In Inspector → RangeChecker → Target field
- Drag the Sphere into that field
 
RESULT: Move the Player with arrow keys.
        Console prints when Player enters or exits range of Target.

FINDING ANGLE TO TARGET
========================
Keep the same scene, same objects. Just add a new script.
 
Step 1 — Create Script
- In Project panel: Right Click → Create → C# Script
- Name it: AngleChecker
 
Step 2 — Script Code
NOTE: Class name must be AngleChecker to match the filename.
 
using UnityEngine;
public class AngleChecker : MonoBehaviour
{
    [SerializeField] Transform target;
    [SerializeField] float moveSpeed = 5f;
 
    void Update()
    {
        // Player movement with arrow keys
        float moveX = Input.GetAxis("Horizontal");
        float moveZ = Input.GetAxis("Vertical");
        Vector3 movement = new Vector3(moveX, 0f, moveZ);
        transform.Translate(movement * moveSpeed * Time.deltaTime, Space.World);
 
        // Angle calculation
        var directionToTarget = (target.position - transform.position).normalized;
        var dotProduct = Vector3.Dot(transform.forward, directionToTarget);
        var angle = Mathf.Acos(dotProduct);
 
        Debug.LogFormat(
            "The angle between my forward direction and {0} is {1:F1}°",
            target.name,
            angle * Mathf.Rad2Deg
        );
    }
}
 
Step 3 — Attach Script
- Select Player (Cube)
- Add Component → AngleChecker
- In Inspector → AngleChecker → Target field
- Drag the Sphere into that field
 
RESULT: Move the Player with arrow keys.
        Console prints the angle between Player and Target every frame.
