Step 1 — Create Scene
- GameObject → 3D Object → Plane   → rename: Ground
- GameObject → 3D Object → Capsule → rename: Player
- GameObject → 3D Object → Capsule → rename: Enemy
 
 
Step 2 — Bake NavMesh
- Select Ground (Plane)
- In Inspector: tick Navigation Static
- Window → AI → Navigation → Click Bake
Now the AI can walk on the plane.
 
 
Step 3 — Enemy Setup
- Select Enemy
- Add Component → NavMesh Agent
This lets the Enemy use Unity pathfinding.
 
 
Step 4 — Player Script
- Create script: Player1
- Attach to Player object
 
using UnityEngine;
public class Player1 : MonoBehaviour
{
    public float speed = 5.0f;
 
    void Update()
    {
        float moveHorizontal = Input.GetAxis("Horizontal");
        float moveVertical   = Input.GetAxis("Vertical");
        Vector3 movement = new Vector3(moveHorizontal, 0.0f, moveVertical);
        transform.Translate(movement * speed * Time.deltaTime, Space.World);
    }
}
 
Controls: WASD or Arrow Keys → Move Player
 
 
Step 5 — Enemy AI Script
- Create script: AIChasePlayer
- Attach to Enemy object
 
NOTE: StopGame() works in both Editor and exported builds.
      The #if block prevents a build error.
 
using UnityEngine;
using UnityEngine.AI;
public class AIChasePlayer : MonoBehaviour
{
    public Transform player;
    public float detectionRange = 10.0f;
    private NavMeshAgent agent;
 
    void Start()
    {
        agent = GetComponent<NavMeshAgent>();
    }
 
    void Update()
    {
        float distance = Vector3.Distance(transform.position, player.position);
 
        if (distance < detectionRange)
        {
            agent.SetDestination(player.position);
            Debug.Log("AI is chasing the player");
 
            if (distance < 0.5f)
            {
                Debug.Log("Enemy Wins!");
                StopGame();
            }
        }
        else
        {
            agent.ResetPath();
        }
    }
 
    void StopGame()
    {
        Debug.Log("Game Over! The enemy caught you.");
        Application.Quit();
#if UNITY_EDITOR
        UnityEditor.EditorApplication.isPlaying = false;
#endif
    }
}
 
 
Step 6 — Link Player to Enemy
- Select Enemy object
- In Inspector → AIChasePlayer → Player field
- Drag the Player object into that field
This tells the AI who to chase.
